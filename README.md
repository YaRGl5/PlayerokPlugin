🤖 GPT Консультант — AI Плагин для Playerok Бот
GPT Consultant — это интеллектуальный модуль (плагин) первой линии поддержки для маркетплейса цифровых товаров Playerok, работающий на базе высокоскоростного Groq API (модели семейства Llama).

Внимание: Этот плагин разработан для стороннего асинхронного Telegram-бота для платформы Playerok. Основная задача модуля — разгрузить продавца от шаблонных вопросов, консультировать покупателей 24/7 и автоматически звать человека только в экстренных ситуациях (споры, неработающие ключи).

🚀 Ключевые возможности
Динамический контекст (База знаний) Плагин не просто пишет от себя. Перед генерацией каждого ответа он подтягивает в свой System Prompt актуальный прайс-лист (
products.json
), FAQ магазина и скрипты продаж. AI знает, что Apple Gift 10$ стоит 950₽, и может рассказать покупателю, как его активировать.
Классификация намерений (Intent Classification) Модуль умеет отличать вопрос "А есть ли карта на 10$?" от жалобы "Здравствуйте, ваш код пишет, что уже активирован!".
Умная Эскалация (Перехват чата) Если покупатель грубит, просит возврат средств или нейросеть определяет проблему "Сложный вопрос / Брак", бот извиняется, пишет "Передаю ваш вопрос живому продавцу, ожидайте" и мгновенно оповещает владельца магазина в Telegram.
Память (Chat History) Плагин запоминает N последних сообщений конкретного покупателя. Вы можете вести с ним диалог, и он не забудет, о каком товаре вы спросили минуту назад.

🛠 Настройки плагина (Конфигурация)
Управление плагином осуществляется прямо из Telegram. Владелец магазина может настраивать:

Таблица параметров	Для чего используется
🔑 API Key	Ключ авторизации Groq (gsk-...).
🤖 Модель	Выбор актуальной модели (например, llama-3.1-8b-instant или llama-3.1-70b-versatile).
📝 Системный Промпт	Глобальная инструкция (роль бота, тон общения, скрипты).
⏱ Задержка (Delay)	Откладывание ответа на 2-5 секунд для имитации "печатания" живого человека.
🚫 Стоп-слова	Массив слов (например: "обман, суд, верните"), при которых AI сразу зовет человека.
🔄 Переключатель	Полное включение/выключение автоответов AI.

3. Если человек жалуется на "уже активированный код" - напиши "Передаю информацию старшему специалисту, ожидайте проверки" и вызови функцию эскалации.
4. Разговаривай приветливо, используй эмодзи.
